== Description ==

Simple Light Box is the simple jquery effect to show a image in lightbox.
**This is THE perfect tool to manage perfectly your WP terms for any taxonomy**

It was written with this philosophy : best performances, more secured and brings a lot of new functions

This plugin is developped on WordPress 3.3, with the constant WP_DEBUG to TRUE.

== Installation ==

**Required PHP5.**

The Simple Tags can be installed in 3 easy steps:

1. Unzip "Simple Light Box" archive and put all files into a folder like "/wp-content/plugins/simple-light-box/"
2. Activate the plugin through the 'Plugins' menu in WordPress
3. As the plugin is activated set class="lightBox" on any image tag which you want to show in lightbox.
